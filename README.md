# Hackathon-Results-Management-System
I built this Hackathon Results Management System as a multi-project .NET 9 console application to demonstrate advanced C# development and database integration. The system features a custom data pipeline that imports project results from XML files, performs strict validation on scores and team sizes, and persists the data into a SQL Server database using Entity Framework Core Code-First migrations.I implemented an event-driven architecture using custom delegates to provide real-time status updates during data processing. The application includes a suite of 15 LINQ queries—ranging from simple filtering to complex grouping and category-based averaging—to analyze performance metrics. All query results are automatically serialized and exported into structured JSON files for reporting.
